# Project Design of Information Systems
This repository contains coursework for the course *Projektiranje informacijskih sustava/Project Design of Information Systems*.
The project focuses on analysis and modeling of an information system, including requirements analysis, process modeling, data modeling, and database design.

## Contents
- Requirements analysis
- System boundary definition
- Process modeling (DFD diagrams)
- Data modeling (ER diagrams)
- Relational database design (tables)
- System design documentation

## Tools Used
- Microsoft Word (documentation)
- Diagramming tools (ER and DFD modeling)
- Database design tools (relational schema / tables)

## Purpose
The goal of this project is to demonstrate understanding of information system design, including analysis, modeling, and database structuring.
